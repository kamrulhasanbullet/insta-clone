# 📸 Instagram Clone
 
A full-stack, production-ready Instagram-like social media application built with **Next.js 15**, **MongoDB**, **Cloudinary**, and **NextAuth**.
 
---
 
## 🚀 Live Demo
 
> https://instagram-clone-dusky-eight.vercel.app/
 
---
 
## ✨ Features
 
### Authentication
- Email/password registration & login
- JWT-based session management via NextAuth
- Protected routes with middleware
 
### Posts
- Create post with image upload (Cloudinary)
- Like / Unlike with optimistic UI
- Comment on posts
- Delete own posts
- Save / Unsave posts
 
### Stories
- Create 24-hour stories
- View stories with auto-progress bar
- Multiple stories per user
- Tap left/right to navigate
- Delete own stories with confirmation modal
 
### Profile
- Profile picture upload
- Bio, website, fullName editing
- Private account toggle
- Followers & Following list modal
- Follow / Unfollow / Follow Back
 
### Feed
- Chronological feed from followed users
- Infinite scroll pagination
- Stories bar at top
- Click avatar to view story
 
### Notifications
- Like, Comment, Follow notifications
- Mark all as read
- Unread count badge
 
### Explore
- Search users by username
- Explore grid of all posts
 
### Saved & Activity
- Save posts and view in Saved page
- Your Activity — liked posts & commented posts
